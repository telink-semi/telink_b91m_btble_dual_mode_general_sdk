en [English]
****************************** load_comp.sh **********************************************
【Function】
	Load the components in the com directory into the corresponding function module of the SDK.
【Operation method】
	1、Double click to run the script.
	2、Prompt "Please enter component names(supports up to 3):" when you enter the name of the
       component you need to load, to load more than one component separated by a space (up to
	   3 components can be loaded at the same time).

****************************** load_comp.sh **********************************************
【功能】
	将com目录下的组件加载到SDK对应的功能模块。
【操作方法】
	1、双击运行脚本
	2、提示“Please enter component names(supports up to 3):”时输入需要加载的组件名，要加载
	   多个组件时用空格隔开（最多可同时加载3个组件）。
