en [English]
****************************** get_project.bat ******************************************
【Function】
	Get the .project and .cproject files of the corresponding chip to the current directory.
【Operation method】
	1、Double click to run the script
	2、Prompt "Please input chip model:" when input B91 or B92, 'B' for uppercase(
	   mandatory case-sensitive)
	3, after the successful implementation of the word 'success' will be displayed,
       at this time you can normally import the project to the Telink IDE!
【Note】
	The script will backup the project files in the root directory to core/B9x/B9x.cproject. 
******************************************************************************************


********************************* rm_output.bat ******************************************
【Function】
	Delete all compiled 'output' folders under the current project.
【Operation method】
	Double click to run the script
******************************************************************************************




zh_CN [简体中文]
****************************** get_project.bat ******************************************
【功能】
	获取对应芯片的.project 和 .cproject文件到当前目录
【操作方法】
	1、双击运行脚本
	2、提示“Please input chip model:”时输入B91或B92，‘B’为大写（强制区分大小写）
	3、执行成功后会显示‘success’字样，此时即可正常导入工程到Telink IDE
【注意事项】
	该脚本会将根目录下的工程文件备份到core/B9x/B9x.cproject 
******************************************************************************************


********************************* rm_output.bat ******************************************
【功能】
	删除当前工程下所有编译产生的‘output’文件夹
【操作方法】
	双击运行脚本
******************************************************************************************
